Dataset Description

Dataset: Spotify_Youtube.csv (30.78 MB)

Collection Date: February 7th, 2023

About Dataset

This dataset contains songs from various artists worldwide, including:

  • Spotify streaming statistics
  • YouTube official music video statistics

Variables

Track: Name of the song, as visible on the Spotify platform

Artist: Name of the artist

Url_spotify: The URL of the artist

Album: The album in which the song is contained on Spotify

Album_type: Indicates if the song is released on Spotify as a single or contained in an album

Uri: A Spotify link used to find the song through the API

Danceability: (0.0-1.0) Describes how suitable a track is for dancing

Energy: (0.0-1.0) Represents intensity and activity

Key: The musical key of the track (integers mapping to pitch class notation)

Loudness: Overall loudness in decibels (dB)

Speechiness: (0.0-1.0) Detects presence of spoken words

Acousticness: (0.0-1.0) Confidence measure of acoustic nature

Instrumentalness: (0.0-1.0) Predicts vocal content

Liveness: (0.0-1.0) Detects audience presence

Valence: (0.0-1.0) Musical positiveness measure

Tempo: Estimated tempo in BPM

Duration_ms: Track duration in milliseconds

Stream: Number of Spotify streams

Url_youtube: YouTube video URL

Title: YouTube video title

Channel: YouTube channel name

Views: YouTube view count

Likes: YouTube like count

Comments: YouTube comment count

Description: YouTube video description

Licensed: YouTube content partner status

official_video: Boolean indicating if it's the official video

Note: These data are heavily dependent on the time they were collected, which is in this case the 7th of February, 2023.
